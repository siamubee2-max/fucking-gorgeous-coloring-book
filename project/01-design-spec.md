# Design Specification — F*cking Gorgeous

## Style Guide
- **Line weight**: 1.5-2pt clean black lines
- **No shading, no grayscale** — pure line art
- **Floral mandala hybrid**: insulte au centre, mandala de fleurs autour
- **Orientation**: Portrait (8.5 x 11")
- **Background**: Blanc pur, pas de texture
- **DPI**: 300 minimum pour impression
