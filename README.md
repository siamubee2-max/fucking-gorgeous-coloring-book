## Workflow

1. Generate images using prompts in `project/02-image-prompts.md`
2. Run `scripts/assemble.sh` to build interior PDF
3. Create cover (see `project/01-design-spec.md`)
4. Publish on KDP (see `KDP.md`)

## Design Specs

- 8.5 x 11 inches, ~114 pages
- Black & white line art, 300 DPI
- Single-sided printing
